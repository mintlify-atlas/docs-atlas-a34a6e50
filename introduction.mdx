# Welcome to PDF AI

PDF AI is an intelligent document assistant that lets you have natural conversations with your PDF files. Upload any PDF and ask questions, get summaries, or extract insights—all powered by advanced AI technology.

## What is PDF AI?

PDF AI combines cutting-edge AI models with vector database technology to understand and respond to questions about your documents. Instead of manually searching through pages, simply ask questions in natural language and get accurate, contextual answers instantly.

<Info>
PDF AI uses **GPT-4** and **Pinecone Vector Database** to deliver fast, accurate responses based on your document's actual content.
</Info>

## How It Works

PDF AI uses a sophisticated RAG (Retrieval-Augmented Generation) architecture:

1. **Upload**: Your PDF is uploaded to secure cloud storage
2. **Processing**: The document is split into chunks and converted to vector embeddings
3. **Storage**: Embeddings are stored in Pinecone vector database for fast retrieval
4. **Query**: When you ask a question, we find the most relevant chunks
5. **Response**: GPT-4 generates an accurate answer using the retrieved context

<Note>
Learn more about the technical architecture in our [How It Works](/how-it-works) guide.
</Note>

## Technology Stack

PDF AI is built with modern, production-ready technologies:

- **Next.js 13** - React framework with App Router
- **OpenAI GPT-4** - Advanced language model for intelligent responses
- **Pinecone** - Vector database for semantic search
- **LangChain** - Framework for PDF processing and document splitting
- **Vercel Edge Functions** - Ultra-fast serverless compute
- **AWS S3** - Secure, scalable file storage
- **Clerk** - Authentication and user management
